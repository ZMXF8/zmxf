使用说明

欢迎使用 ZMXF 网络检站，本页面可检测您的公网 IP、运营商、国家、城市、经纬度等详细信息。

功能列表

  • 显示真实公网 IP 和物理地址
  • 自动识别国家、省份、城市、邮编、ISP、ASN
  • 获取设备信息：语言、浏览器 UA
  • 截图保存：点击一键截图按钮

使用方法

  1. 直接访问 index.html 页面
  2. 页面加载后自动获取网络信息
  3. 点击“📸 一键截图保存”按钮保存页面信息（建议使用浏览器截图插件）

部署到 GitHub Pages

将整个项目文件上传到你的 GitHub 仓库，启用 Pages 功能，选择部署根目录，即可免费上线。
